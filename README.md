# League of Legends Statistics Discord Bot

## Description
A Python-based Discord bot that collects player statistics from the Riot Games API and sends detailed reports to Discord channels.

## Features
- Fetches player data using Riot API
- Tracks player statistics
- Formats and sends reports to Discord
- Supports multiple players via stored PUUIDs

## Technologies
- Python
- Riot Games API
- Discord API

## Project Structure
- discord_bot.py – Discord bot logic
- Stats_track.py – statistics processing
- players.json / puuids.json – player data
- test.py – testing utilities

## Setup
1. Install required dependencies
2. Configure Riot API key
3. Run the Discord bot
